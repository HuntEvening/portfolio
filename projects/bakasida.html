Bootstrap and LESS

This assignment consisted of with the help of a framework of our choice create a website that has all its CSS generated with LESS or SASS. The layout is free of choice but needs to contain 5 pages and 2 secondary pages. The CSS files for the framework is not allowed to be edited. I chose bootstrap as my framework since I have been working a lot with it before and is comfortable with using it. I had a hard time choosing between SASS and LESS since they are similar but decided I would start with LESS and work with SASS on a future project. I had found some pictures that inspired me to create a baking site combined with a blog, and that way I could also show off the best way of using the grid system in bootstrap.

Course: Advanced web development with HTML and CSS
